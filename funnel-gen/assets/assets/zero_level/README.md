This folder contains JSON data for the zero-level generator.

Files:
- species.json — list of available species with languages and optional weights
- backgrounds.json — list of backgrounds with skills/tools/languages
- proficiencies.json — global default weapon/tool proficiencies for level 0
- gear.json — example gear kits to randomly assign
- hit_points.json — HP rule (dice + whether to add CON modifier)

Note: These are starter examples and should be expanded to match the PDF tables you use. Update without code changes.
